# 天佑电影数据库项目

## 项目介绍
一个基于 Spring Boot 的电影数据库管理系统，用于管理和展示电影信息。

### 主要功能
- 电影信息管理（添加、修改、删除、查询）
- 用户管理系统
- 电影分类管理
- 评分和评论功能
- 电影海报和预告片展示

### 技术栈
- 后端：Spring Boot + MyBatis
- 数据库：MySQL
- 前端：Thymeleaf + Bootstrap
- 构建工具：Maven

## 项目规范
### 代码规范
- 类名：首字母大写的驼峰命名
- 方法名：首字母小写的驼峰命名
- 变量名：驼峰命名
- 常量名：全大写，下划线分隔

### Git 提交规范
- feat: 新功能
- fix: 修复bug
- docs: 文档更新
- style: 代码格式化
- refactor: 代码重构
- test: 测试相关
- chore: 构建过程或辅助工具的变动
